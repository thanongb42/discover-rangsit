📋 คู่มือการโพสต์ร้านบน Discover Rangsit

ทำตามขั้นตอนง่ายๆ ภายใน 5 นาที — ฟรี ไม่มีค่าใช้จ่าย

1
เข้าเว็บไซต์ Discover Rangsit เริ่มต้น

เปิดเบราว์เซอร์แล้วไปที่ URL ด้านล่าง หรือสแกน QR Code

  • พิมพ์ discover.rangsitcity.go.th ในแถบ URL
  • กดปุ่ม "เข้าสู่ระบบ" ที่มุมขวาบนของหน้าเว็บ
2
สมัครสมาชิก / เข้าสู่ระบบ ต้องมีบัญชี

เลือกวิธีเข้าสู่ระบบที่สะดวกที่สุดสำหรับคุณ

  • LINE Login — กดปุ่มสีเขียว "เข้าสู่ระบบด้วย LINE" อนุญาตแล้วระบบ Login อัตโนมัติ
  • Google Login — กดปุ่ม "เข้าสู่ระบบด้วย Google" เลือกบัญชี Gmail
  • Email/Password — กรอกอีเมลและรหัสผ่าน หรือกด "สมัครสมาชิก" ถ้ายังไม่มีบัญชี
💡 แนะนำให้ใช้ LINE Login เพราะสะดวกและรวดเร็วที่สุด ไม่ต้องจำรหัสผ่าน
3
ไปที่ Dashboard แล้วกด "เพิ่มสถานที่" เพิ่มร้าน

หลัง Login แล้วจะเห็นเมนูด้านบน กดที่ชื่อของคุณ → "Dashboard"

  • กดปุ่ม "+ เพิ่มสถานที่ใหม่" ในหน้า Dashboard
  • หรือกดเมนู "เพิ่มสถานที่" ในแถบด้านบน
4
กรอกข้อมูลร้านของคุณ ข้อมูลสำคัญ

กรอกข้อมูลให้ครบถ้วน ยิ่งครบยิ่งดี ลูกค้าจะค้นหาเจอง่ายขึ้น

  • ชื่อร้าน / สถานที่ — กรอกชื่อที่คนรู้จัก
  • หมวดหมู่ — เลือกประเภทของร้าน เช่น ร้านอาหาร, คาเฟ่, ร้านค้า
  • รายละเอียด — เขียนอธิบายร้านของคุณ สินค้า บริการ จุดเด่น
  • ที่อยู่ — กรอกที่อยู่ให้ชัดเจน เพื่อแสดงบนแผนที่
  • เบอร์โทร — ลูกค้าจะได้ติดต่อได้ง่าย
  • รูปภาพหน้าปก — อัปโหลดรูปสวยๆ ของร้านคุณ
  • โซเชียล — Facebook, LINE, Instagram (ถ้ามี)
📸 รูปภาพสำคัญมาก! ร้านที่มีรูปสวยจะได้รับความสนใจมากกว่าถึง 3 เท่า แนะนำรูปขนาดอย่างน้อย 800x600px
5
ปักหมุดตำแหน่งบนแผนที่ แผนที่

ช่วยให้ลูกค้าหาร้านคุณเจอบน City Map ได้ง่ายขึ้น

  • กดที่แผนที่เพื่อปักหมุดตำแหน่งร้านของคุณ
  • หรือกรอก Latitude / Longitude ถ้าทราบพิกัด
  • ลากหมุดเพื่อปรับตำแหน่งให้ถูกต้อง
6
กดส่ง รอการอนุมัติ เกือบเสร็จแล้ว!

กดปุ่ม "ส่งเพื่อรอการอนุมัติ" แล้วรอทีมงานตรวจสอบ

  • ทีมงานจะตรวจสอบข้อมูลภายใน 1-3 วันทำการ
  • เมื่ออนุมัติแล้ว ร้านของคุณจะแสดงบนเว็บทันที
  • ดูสถานะได้ที่ Dashboard → "ร้านของฉัน"
⏳ ถ้าข้อมูลครบถ้วนและถูกต้อง ทีมงานจะอนุมัติให้เร็วที่สุด ข้อมูลที่ไม่เหมาะสมอาจถูกปฏิเสธ

🎉 ยินดีด้วย! ร้านของคุณพร้อมให้คนค้นหาแล้ว

ร้านของคุณจะแสดงบนหน้าหลัก, แผนที่ City Map และผลการค้นหา
แชร์ลิงก์ร้านของคุณให้ลูกค้าและเพื่อนๆ ได้เลย!
