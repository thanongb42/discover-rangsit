  • 📍 ค้นหาร้านใกล้ฉันระบุตำแหน่ง GPS หาร้านใกล้เคียง
  • 🌤️ ข้อมูล Realtimeสภาพอากาศ + ค่าฝุ่น PM2.5

วิธีลงทะเบียน

6 ขั้นตอน เพิ่มร้านของคุณ

1

เข้าเว็บไซต์

เปิด discover.rangsitcity.go.th บนมือถือหรือคอมพิวเตอร์

2

ล็อกอิน

กดปุ่ม "เข้าสู่ระบบ" เลือก LINE, Google หรือสมัครด้วย Email

3

แดชบอร์ด → เพิ่มธุรกิจ

กดเมนูชื่อตัวเอง เลือก "แดชบอร์ด" แล้วกด "เพิ่มธุรกิจ"

4

กรอกข้อมูลร้าน

ชื่อร้าน หมวดหมู่ รายละเอียด เบอร์โทร โซเชียลมีเดีย และรูปภาพ

5

ปักหมุดบนแผนที่

กดปุ่ม "ตำแหน่งปัจจุบัน" หรือเลือกจุดบนแผนที่ด้วยตนเอง

6

ส่งรอการอนุมัติ

กด "ส่งข้อมูล" รอเจ้าหน้าที่ตรวจสอบ 1–3 วันทำการ

กรอกข้อมูลอย่างไร
ให้ได้ผลดีที่สุด?

  • 📸 รูปภาพสำคัญที่สุด ใช้รูปคมชัด แสงดี ขนาด > 800×600px ร้านที่มีรูปสวยได้รับความสนใจมากกว่า
  • ✍️ รายละเอียดชัดเจน อธิบายสินค้า/บริการ เวลาเปิด-ปิด จุดเด่นของร้าน
  • 📍 ปักหมุดให้ถูกต้อง กด "ตำแหน่งปัจจุบัน" ขณะอยู่ที่ร้าน หรือเลือกจุดบนแผนที่
  • 💚 ใส่ LINE ID ของร้าน ลูกค้าจะสแกน QR Code Add LINE ร้านได้โดยตรง

ข้อมูลที่กรอกได้

  • ✓ ชื่อร้านและหมวดหมู่
  • ✓ รายละเอียดธุรกิจ / เมนู / บริการ
  • ✓ ที่อยู่และพิกัด GPS
  • ✓ เบอร์โทร + เว็บไซต์
  • ✓ LINE / Facebook / Instagram / TikTok
  • ✓ รูปหน้าปก + QR Code LINE + Gallery
